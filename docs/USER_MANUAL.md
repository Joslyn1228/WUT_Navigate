## 快速开始

### 系统要求
- 现代浏览器（Chrome、Firefox、Safari、Edge）
- 移动端建议使用微信内置浏览器或系统浏览器
- 需要开启位置权限以使用导航和讲解功能

### 访问方式
1. 打开浏览器访问：`http://localhost:8080`
2. 或扫描二维码访问（部署后）

### 账号注册
1. 点击右上角"注册"按钮
2. 填写用户名、密码、邮箱、昵称
3. 点击"注册"完成账号创建

### 账号登录
1. 点击右上角"登录"按钮
2. 输入用户名和密码
3. 点击"登录"进入系统

## 导航功能

### 如何使用导航
1. 点击页面顶部"导航"标签
2. 点击定位图标（📍）获取当前位置（需要位置权限）
3. 在搜索框输入目的地，如"南湖图书馆"、"南湖体育馆"
4. 选择合适的目的地
5. 选择导航模式：
   - 🚶‍♂️ 步行导航
   - 🚲 骑行导航
   - 🚌 校车导航
6. 点击"开始导航"
7. 地图上会显示路线，页面显示步骤指引

### 导航模式说明
| 模式 | 说明 | 适用场景 |
|------|------|----------|
| 步行 | 步行路线，优先校内道路 | 短距离出行 |
| 骑行 | 骑行路线，避开步行区域 | 校区内骑行 |
| 校车 | 校车路线，按站点导航 | 校区间往返 |

### 注意事项
- 确保已开启位置权限
- 导航目的地仅限武汉理工大学校内
- 实时导航请保持屏幕常亮

## 实时讲解

### 自动讲解
1. 点击"实时讲解"标签
2. 点击定位图标（📍）获取当前位置
3. 系统自动监控位置变化
4. 当您接近景点（默认20米范围内）时会自动弹出讲解

### 手动查看讲解
1. 在景点列表中选择想了解的景点
2. 点击景点卡片查看详情
3. 可选择以下讲解方式：
   - 📖 文字阅读
   - 🔊 语音播放
   - 📹 视频讲解（如提供）
   - 🖼️ 图片展示

### 收藏景点
1. 在景点详情页点击⭐收藏按钮
2. 可在"我的收藏"中查看已收藏景点
3. 再次点击可取消收藏

### 分享讲解
1. 在景点详情页点击🔗分享按钮
2. 选择分享方式：
   - 复制链接
   - 分享到微信
   - 分享到朋友圈

## 打卡功能

### 如何打卡
1. 到达景点附近（需要在电子围栏范围内）
2. 点击"打卡"按钮
3. 确认景点信息后点击"确认打卡"
4. 打卡成功！可能获得新成就

### 查看打卡历史
1. 点击"我的"标签
2. 选择"打卡历史"
3. 可按时间查看所有打卡记录
4. 点击记录可查看详情

### 成就系统
| 成就名称 | 获取条件 | 图标 |
|----------|----------|------|
| 首次打卡 | 完成第一次打卡 | 🎯 |
| 打卡达人 | 累计打卡10次 | 🏆 |
| 打卡大师 | 累计打卡50次 | 👑 |
| 坚持一周 | 连续打卡7天 | 📅 |
| 收藏家 | 收藏5个景点 | 📚 |

### 查看成就
1. 点击"我的"标签
2. 选择"我的成就"
3. 可查看已获得的成就和待解锁的成就

### 打卡统计
在"打卡统计"页面可查看：
- 总打卡次数
- 连续打卡天数
- 打卡过的景点数量
- 各景点打卡次数

## 运动健康

### 开始运动
1. 点击"运动"标签
2. 选择运动类型：
   - 🚶‍♂️ 步行
   - 🏃‍♂️ 跑步
   - 🚲 骑行
3. 点击"开始运动"
4. 系统开始记录运动轨迹

### 运动中
- 实时显示运动距离
- 实时显示运动时长
- 实时显示当前位置
- 实时估算消耗卡路里

### 结束运动
1. 点击"结束运动"
2. 确认运动数据
3. 点击"保存"完成记录
4. 可查看运动总结

### 运动数据
在"运动统计"页面可查看：
- 今日数据
- 本周数据
- 本月数据
- 历史记录

数据包括：
- 总运动距离
- 总运动时长
- 总消耗卡路里
- 运动次数
- 平均速度

## 常见问题

### Q: 定位失败怎么办？
A:
1. 检查是否开启了位置权限
2. 确保手机GPS已打开
3. 尝试在户外开阔地带获取位置
4. 刷新页面重试

### Q: 导航不准确怎么办？
A:
1. 确认GPS信号良好
2. 检查是否选择了正确的目的地
3. 尝试重新规划路线
4. 如有问题请反馈给我们

### Q: 打卡失败怎么办？
A:
1. 确认已到达景点附近（20米范围内）
2. 检查GPS信号
3. 确认该景点是打卡点
4. 重试打卡

### Q: 语音播放没声音？
A:
1. 检查手机音量是否开启
2. 确认没有静音
3. 检查浏览器是否允许自动播放
4. 手动点击播放按钮

### Q: 如何导出运动数据？
A:
1. 进入运动历史
2. 选择需要导出的记录
3. 点击分享或导出按钮
4. 选择导出格式（如CSV、PDF）

### Q: 如何删除账号？
A:
1. 进入个人设置
2. 选择"账号与安全"
3. 点击"注销账号"
4. 按照提示完成操作

### Q: 遇到Bug怎么办？
A:
1. 截图保存错误信息
2. 记录错误发生的步骤
3. 点击"我的"→"意见反馈"
4. 描述问题并上传截图
5. 我们会尽快处理

### Q: 有新功能建议？
A:
欢迎通过以下方式反馈：
1. 意见反馈功能
2. 发送邮件至：support@whut.edu.cn
3. 在社区发布建议动态
